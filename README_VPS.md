# euserv德鸡自动续期

vps部署教程
* [ACTION部署](./README.md)

## 1.主要功能
   
   该脚本实现一键脚本直接安装，支持docker部署和python本地部署（注意：配置较低的VPS选择python本地部署，比如使用德鸡vps安装时）

## 2.vps部署

   一键脚本，快捷方式：dj
   ```bash
   wget https://raw.githubusercontent.com/dufei511/euserv_py/dev/install.sh -O euserv_install.sh
   chmod +x euserv_install.sh
   ./euserv_install.sh install
   ```

   
   1.一键脚本执行后，分别需要输入euserv登录邮箱、euserv登录密码、邮箱的应用专用密码，然后后面推送参数都是选输
